Termos de Uso e Política de Privacidade - LGPD

Importante: Este sistema foi desenvolvido em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018).

1. Coleta e Uso de Dados

Coletamos apenas os dados necessários para o processamento das denúncias:

  • Denúncia Anônima: Apenas os dados da denúncia e metadados técnicos (IP, user agent)
  • Denúncia Identificada: Dados da conta do usuário e dados de contato opcionais

2. Finalidade do Tratamento

Os dados coletados são utilizados exclusivamente para:

  • Processar e analisar denúncias de violações da LGPD
  • Manter histórico para fins legais e regulatórios
  • Comunicar-se com o denunciante quando necessário

3. Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros, exceto:

  • Por determinação judicial ou requisição de autoridade competente
  • Para cumprimento de obrigação legal ou regulatória

4. Direitos do Titular

Conforme a LGPD, você tem direito a:

  • Confirmar a existência de tratamento de dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Anonimizar, bloquear ou eliminar dados desnecessários
  • Eliminar dados tratados com consentimento
  • Revogar o consentimento a qualquer momento

5. Segurança dos Dados

Implementamos medidas de segurança técnicas e administrativas para proteger seus dados contra:

  • Acessos não autorizados
  • Situações acidentais ou ilícitas
  • Perda, alteração, comunicação ou destruição

6. Retenção de Dados

As denúncias são mantidas pelo prazo necessário para:

  • Cumprimento de obrigações legais
  • Exercício regular de direitos
  • Preservação de provas e histórico

7. Consentimento

Ao utilizar este sistema, você concorda com:

  • Coleta e tratamento de dados conforme esta política
  • Uso de cookies essenciais para funcionamento do sistema
  • Comunicações relacionadas ao andamento das denúncias
Encarregado de Proteção de Dados (DPO):
Para exercer seus direitos ou esclarecer dúvidas sobre proteção de dados, entre em contato com nosso Encarregado através do e-mail: dpo@denunciaslgpd.com
